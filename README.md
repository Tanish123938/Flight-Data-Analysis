# Flight Data Analysis – Python Project

## Project Overview

This Python project focuses on analyzing flight data to uncover insights related to flight prices, airlines, routes, travel duration, and trends. The analysis helps understand patterns in air travel and supports data-driven decision-making for pricing and planning.

## Objectives

- Analyze flight price trends
- Compare airlines based on pricing and frequency
- Study route-wise and duration-based patterns
- Practice real-world data analysis using Python

## Key Features

- Exploratory Data Analysis (EDA) using Python
- Airline-wise Price Comparison
- Route-wise Analysis (source → destination)
- Duration & Stops Analysis
- Price Distribution & Trends
- Data Cleaning & Preprocessing for accuracy
